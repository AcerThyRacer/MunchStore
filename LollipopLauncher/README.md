# LollipopLauncher

A candy-themed Android launcher with Lollipop aesthetics, integrated with SugarMunch effects.

## Features

- 🍭 Lollipop-inspired Material Design UI
- 🎨 Customizable themes and icons
- ⚡ Fast and lightweight
- 🎯 Integration with SugarMunch effects system
- 📱 Smooth animations and transitions

## Build

```bash
./gradlew assembleDebug
```

## Integration

This launcher is part of the SugarMunch ecosystem and supports:
- SugarBeats haptic feedback
- SugarGallery theme integration
- System-wide FAB overlay
- All SugarMunch visual effects
