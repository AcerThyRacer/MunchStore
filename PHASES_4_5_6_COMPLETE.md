## 🚀 HOW TO USE NEW FEATURES

### 1. Custom Theme Builder
```kotlin
// Navigate to Theme Builder
navController.navigate("theme-builder")

// Save custom theme
viewModel.saveTheme()

// Share theme
val shareCode = theme.shareCode // "SM-ABC123"
```

### 2. Custom Effect Builder
```kotlin
// Navigate to Effect Builder
navController.navigate("effect-builder")

// Combine effects
viewModel.addEffect(EffectV2.DRAGON_BREATH)
viewModel.addEffect(EffectV2.STARDUST_TRAIL)

// Save custom effect
viewModel.saveEffect()
```

### 3. Cloud Backup
```kotlin
// Enable backup
backupManager.setBackupEnabled(true)

// Backup themes
backupManager.backupThemes(themesJson)

// Restore
backupManager.restoreThemes()
```

### 4. Music-Reactive Themes
```kotlin
// Start listening
musicReactiveTheme.startListening(audioSessionId)

// Get reactive colors
val colors = musicReactiveTheme.getReactiveGradient()

// Stop when done
musicReactiveTheme.stopListening()
```

### 5. Weather-Reactive Themes
```kotlin
// Start tracking
weatherReactiveTheme.startWeatherTracking()

// Get weather theme
val colors = weatherReactiveTheme.getWeatherThemeColors()
```

### 6. Theme Scheduler
```kotlin
// Enable auto-switch
themeScheduler.setAutoSwitchEnabled(true)

// Add schedule
themeScheduler.addScheduledTheme(scheduledTheme)

// Apply preset
themeScheduler.applyPresetSchedule(preset)
```

### 7. Social Features
```kotlin
// Log activity
socialRepository.logActivity(
    type = ActivityType.INSTALLED_APP,
    targetType = TargetType.APP,
    targetId = appId,
    targetName = appName
)

// Follow user
socialRepository.followUser(userId)

// Share theme
socialRepository.shareTheme(communityContent)
```

### 8. Seasonal Events
```kotlin
// Update active event
eventManager.updateActiveEvent()

// Get progress
val progress = eventManager.getEventProgress()

// Claim reward
eventManager.claimReward(rewardId)
```
